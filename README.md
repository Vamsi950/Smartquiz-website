# Smart Quiz

An intelligent quiz platform built with modern web technologies.

## Project Structure

- **QuizCraft/frontend**: React-based frontend application.
- **QuizCraft/backend**: Node.js/Express backend server.

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Vamsi950/Smartquiz-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd "Smart Quiz/QuizCraft"
   ```

### Running the App
- **Backend**:
  ```bash
  cd backend
  npm install
  npm start
  ```
- **Frontend**:
  ```bash
  cd frontend
  npm install
  npm run dev
  ```

## Features
- Dynamic quiz generation
- Admin dashboard for analytics and management
- Performance tracking and certificates
